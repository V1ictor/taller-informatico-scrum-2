# Gestión del Taller Informático
## Historias de Usuario (Product Backlog)

1. **Recepción:** Como recepcionista, quiero registrar la entrada de equipos y generar un ID único.
2. **Datos Cliente:** Como recepcionista, quiero capturar datos del cliente y descripción del problema.
3. **Diagnóstico:** Como técnico, quiero registrar síntomas y generar informe de diagnóstico.
4. **Presupuesto:** Como administrativo, quiero crear presupuestos detallados (mano de obra/piezas).
5. **Aprobación:** Como administrativo, quiero registrar la aceptación/rechazo del presupuesto.
6. **Reparación:** Como técnico, quiero registrar avances y finalización de la reparación.
7. **Stock:** Como encargado, quiero gestionar inventario y pedir piezas.
